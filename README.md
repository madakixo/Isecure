# SecureEye-NG v19 - Ultimate Scalable Face Recognition PWA

Real-time face detection & recognition using YOLOv8 + InsightFace  
Supports 1 to 100+ cameras · 100,000+ concurrent users · Full PWA (installable on phone/desktop)

## Features
- Ultra-fast YOLOv8-face + InsightFace recognition
- 1–10+ RTSP cameras simultaneously
- Live view in browser (mobile & desktop)
- Installable PWA with offline mode
- Auto-save unknown faces
- Google Drive backup (optional)
- Horizontal scaling (Docker Swarm / Kubernetes ready)
- HTTPS with auto Let's Encrypt
